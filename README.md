# Credit Risk Prediction

## Overview
This project analyzes customer financial data to identify factors influencing credit risk and predict loan default probability using machine learning techniques.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Key Features
- Exploratory Data Analysis (EDA)
- Data Cleaning & Preprocessing
- Feature Importance Analysis
- Predictive Modeling

## Business Objective
Support data-driven lending decisions by identifying high-risk customers based on financial indicators.
